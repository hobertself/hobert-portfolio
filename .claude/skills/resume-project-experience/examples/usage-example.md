## 示例2：使用skill的完整流程

1. **触发条件**：用户明确要求将项目文档转化为简历项目经历
2. **文档读取**：使用Read工具读取Word文档内容
3. **内容分析**：提取项目背景、职责、成果等关键信息
4. **STAR法则应用**：按STAR法则组织内容
5. **四字词语匹配**：从词库选择合适的四字词语
6. **量化成果**：提取并量化关键成果
7. **输出格式**：生成最终的简历项目经历

## 支持的文件格式
- .docx (Word文档)
- .doc (旧版Word)
- .pdf (PDF文件)
- .md (Markdown文件)
- .txt (文本文件)

## 注意事项
- 确保文件路径正确
- 文件格式受支持
- 文件内容应包含项目相关信息
