# 实习经历转化 Skill

## 功能概述

从实习文档中读取信息，转化为高质量的简历实习经历，采用STAR法则结构，每部分前用四字词语概括，突出量化成果。

## 核心功能

1. **STAR法则结构化**
   - 背景（Situation）：描述工作背景和任务
   - 职责（Task）：说明具体职责和目标
   - 行动（Action）：陈述采取的行动，动词开头
   - 成果（Result）：量化成果和影响

2. **四字词语凝练**
   - 项目背景：凝练项目背景和价值
   - 核心职责：概括主要工作内容
   - 实施行动：说明具体工作方法
   - 成果效益：量化工作成果价值

3. **数据量化成果**
   - 提取可量化的数据指标
   - 强调具体数字和对比效果
   - 突出个人贡献和价值

## 工作流程

1. **读取解析**：提取关键信息，识别量化数据
2. **STAR构建**：按背景→职责→行动→成果组织内容
3. **四字概括**：为每部分添加四字词语凝练
4. **数据强化**：为所有成果添加量化数据
5. **精简优化**：去除冗余，突出核心价值

## 输出示例

**关键字：市场调研、消费者洞察、数据分析、Python、SPSS**

```
# 实习经历：职位名称 | 公司名称 | 时间

**市场洞察**：国内滑板运动快速发展但本土品牌市场潜力未充分释放。针对天津市滑板爱好者开展深度调研，为Justice品牌提供市场定位与营销策略支持。

**核心职责**：负责问卷架构设计、多渠道数据收集及定量数据建模分析，运用专业统计方法挖掘消费者购买影响因素。

**数据分析**：运用Python爬虫抓取电商平台超百条评论，通过文本挖掘生成词云图；采用线上线下结合方式发放问卷，建立三级审核机制，有效回收393份问卷；运用SPSS进行NPS模型分层、卡方检验、主成分分析和Logistic回归建模。

**成果效益**：有效问卷回收率达88.71%，数据信效度优良（Cronbach's α系数>0.9）；构建"感知质量"预测模型，回归系数0.453（P=0.006），验证其对购买行为的显著正向影响；输出60页商业调查报告，提出6大落地策略，助力项目获天津赛区二等奖。
```

## 质量控制

### 确认要点
1. **信息准确性**：确保所有数据、时间、职责真实准确
2. **STAR完整性**：严格遵循背景→职责→行动→成果结构
3. **四字凝练**：每部分前用四字词语概括核心要点
4. **数据量化**：所有成果必须有具体数据支撑
5. **简洁精炼**：去除冗余内容，突出核心价值

### 四字词语库

### 项目背景
市场洞察、行业背景、项目起因、业务需求、现状分析、问题挑战

### 核心职责
工作内容、主要职责、任务范围、负责领域、工作目标、职责范围

### 实施行动
数据分析、市场调研、方案设计、技术实施、策略制定、流程优化

### 成果效益
量化成果、效益提升、价值创造、业绩达成、成果展示、项目影响

## 量化表达模板

- 效率提升：效率提升X%，从X优化至Y
- 质量提升：质量提升X%，达到Y标准
- 成果数量：完成X份问卷/报告，回收率达X%
- 数据分析：分析X份数据，发现X个关键因素
- 获得认可：获得X奖项，获得X好评
- 商业价值：创造X策略，实现X效益
