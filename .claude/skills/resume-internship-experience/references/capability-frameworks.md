# 能力框架参考

## 核心能力维度

### 1. 专业技能维度
- **行业知识**：对行业的理解和认知
- **技术能力**：专业技能的掌握程度
- **工具使用**：相关工具的熟练度
- **专业知识**：专业理论和基础知识

### 2. 解决问题维度
- **分析能力**：分析问题和信息的能力
- **逻辑思维**：逻辑推理和判断能力
- **创新能力**：提出创新解决方案
- **决策能力**：做出合理判断和选择

### 3. 沟通协作维度
- **表达能力**：清晰表达思想和观点
- **倾听能力**：理解他人需求和意见
- **团队协作**：与团队成员有效合作
- **关系建立**：建立和维护良好关系

### 4. 学习成长维度
- **学习能力**：快速学习新知识技能
- **适应能力**：适应变化和新环境
- **反思能力**：总结经验教训
- **自我驱动**：主动学习和进步

### 5. 执行力维度
- **计划能力**：制定和执行计划
- **组织能力**：组织和协调资源
- **效率意识**：注重效率和效果
- **结果导向**：关注最终成果

## 能力等级描述

### 初级（1-2级）
- **描述**：能够完成基础性工作，在指导下完成任务
- **表现**：理解基本概念，执行明确指令，需要少量指导
- **关键词**：学习、协助、执行、基础、了解

### 中级（3-4级）
- **描述**：能够独立完成工作，解决常见问题
- **表现**：熟练应用技能，自主处理任务，能应对复杂情况
- **关键词**：独立、熟练、解决、负责、优化

### 高级（5级以上）
- **描述**：能够处理复杂问题，指导他人，推动创新
- **表现**：深度专业能力，系统思考，创新解决方案，团队领导
- **关键词**：精通、创新、领导、战略、突破

## 能力映射方法

### 1. 从工作内容映射
```
工作内容：负责数据分析报告
↓
相关能力：
- 数据处理能力（专业技能）
- 分析能力（解决问题）
- 报告撰写（沟通表达）
```

### 2. 从成果影响映射
```
成果：提升转化率30%
↓
体现能力：
- 数据分析能力（从数据中发现机会）
- 优化能力（提出并实施优化方案）
- 结果导向（关注实际效果）
```

### 3. 从工作难度映射
```
任务复杂度：跨部门项目协调
↓
能力等级：
- 沟通协调能力（中级）
- 项目管理能力（中级+）
- 资源整合能力（高级）
```
